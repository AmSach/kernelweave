# FAQ

## Is this just a prompt wrapper?
No. It is a memory and routing architecture around verified execution kernels.

## Does it replace the base model?
No. It makes the base model more useful on repeated work.

## Is it just RAG?
No. RAG retrieves documents. KernelWeave retrieves reusable verified task behavior.

## Is the current runnable path full neural training?
No. The runnable path is the pure-Python calibration/tracing prototype.
