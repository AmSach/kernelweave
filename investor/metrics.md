# Metrics

## Primary metrics
- token reduction vs baseline
- latency reduction vs baseline
- verification pass rate
- kernel reuse rate
- routing precision
- routing recall
- repeated-task coverage
- cost per task family

## Demo benchmark tasks
- compare files
- summarize diffs
- extract structured data
- generate tests
- code review suggestions
- changelog drafting

## Success definition
A repeated task should get cheaper and more reliable after KernelWeave learns it.
