# NeurIPS submission guide for KernelWeave

## What NeurIPS wants
From the 2025 CFP and checklist:
- one PDF submission
- main text within **9 content pages**
- references, checklist, and technical appendix can be after the main text
- checklist must be included
- no identifying info for double-blind review
- all authors need an OpenReview profile by the full paper deadline
- code/supplementary can be in a separate ZIP, anonymized
- if using LLMs in the method, declare it
- you should include limitations and broader impacts discussion
- if you ran experiments, explain reproducibility, settings, resources, and significance appropriately

## What KernelWeave should submit
- `paper/main.tex`
- `kernelweave-neurips-paper.pdf`
- `kernelweave-neurips-codebase.zip`
- optional appendix PDF if you want it separate

## Step-by-step checklist
1. Make sure the main paper is 9 pages or less.
2. Keep the appendix after references or in the supplemental ZIP.
3. Remove identifying acknowledgements from the submission version.
4. Ensure the PDF includes the NeurIPS checklist.
5. Make sure the code zip contains tests, docs, and build instructions.
6. Verify every author has an updated OpenReview profile.
7. Upload the paper to OpenReview before the abstract deadline.
8. Upload any supplementary ZIP before the supplementary deadline.
9. Double-check the title/abstract do not contain placeholder language.
10. Confirm the paper obeys the NeurIPS code of conduct and dual submission policy.
