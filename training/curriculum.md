# Curriculum

## Stage 0: sanity
- tokenization
- config validation
- tiny overfit checks

## Stage 1: language base
- general text
- code
- math

## Stage 2: reasoning
- structured reasoning traces
- stepwise decomposition
- self-consistency style examples

## Stage 3: tool use
- agent traces
- recoverable failure cases
- tool planning and verification

## Stage 4: long context
- retrieval
- compression
- multi-hop memory tasks

## Stage 5: skill bank
- promote successful traces into kernels
- re-train on kernel reuse examples
- demote stale skills

## Stage 6: evaluation hardening
- adversarial prompts
- drift tests
- regression gates
