# KernelWeave quick use

## Files
- `file 'KernelWeave_LLM_Product_Train.py'` — full training + benchmark + realtime demo runner
- `file 'KernelWeave_Demo_Guide.md'` — interview pack and talking points

## Fast path
1. Run the Kaggle command from the guide.
2. Open:
   - `kernelweave_llm_bundle/benchmark/benchmark.md`
   - `kernelweave_llm_bundle/realtime_demo.md`
   - `kernelweave_llm_bundle/adapter/`
   - `kernelweave_llm_bundle/merged_model/`
3. Zip file to share: `kernelweave_llm_export/kernelweave_llm_bundle.zip`
